Getting started and connected to Gumnut should only take a few minutes.

# Installation

```bash
npm install gumnut-v0-api
```

## Quick Start

```javascript
import { connectToGumnut, buildTestToken } from 'gumnut-v0-api';
import 'gumnut-v0-api/dom';

// Create an AbortController to manage the connection lifecycle
const controller = new AbortController();

// Connect to Gumnut
const gumnut = connectToGumnut(controller.signal, {
  projectId: 'your-project-id'
});

// Provide authentication token
const token = buildTestToken('user-123', { name: 'User Name', email: 'user@example.com' });
gumnut.provideToken(token);

// Join a document
const doc = gumnut.join(controller.signal, 'doc-123');

// Use the document with a Gumnut component
document.querySelector('gumnut-textarea').document = doc.forNode('text-content');
```
In your HTML you'll need to use a `<gumnut-textarea>` element instead of a regular `<textarea>`.

```html
<gumnut-textarea></gumnut-textarea>
```

That should be it! You should now be connected to Gumnut and ready to start collaborating.

Ready to get started? Check out the [Getting Started guide](/guides/getting-started) for a more detailed walkthrough.
